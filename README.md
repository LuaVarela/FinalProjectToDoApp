🛠 Project setup:
Basic setup for Vue.js and Supabase:
$ npm vite create "my-todo-project"
$ cd my-todo-project
$ npm add @supabase/supabase-js
$ npm add pinia
$ npm add vue-router
$ npm add pinia-plugin-persist
$ npm i bootstrap@5.2.2
